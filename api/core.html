static generateSliceBinaries(component, root)¶
参数

  • component (Component) – 待拆部件

  • root (Component) – 根部件

返回类型

List[int]

返回

根部件在待拆部件中的所有有效切片。没有找到切片时返回空列表。

找出一个字根在一个部件中的所有有效切片。

例: 待拆部件「夫」，笔画列表为`[‘横’, ‘横’, ‘撇’, ‘捺’]`， 其每一位都取用 index 列表表示为`[0, 1, 2, 3]`，二进制表示为0b1111，即十进制数字15。 根部件「大」，其在「夫」中的有效的切片为「夫」的第1、3、4笔，或第2、3、4笔， 其二进制表示分别为 0b1011、0b0111，即十进制数字 11、7。 故输出有效切片集为`[11, 7]`。
