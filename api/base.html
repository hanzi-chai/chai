class pychai.base.Linear(p0, p1)¶

一次 Bezier 曲线（即直线）的形式为：

\[\r_1(t)=(1-t)\p_1+t\p_2\]

它的几何直观是：\(t=0\) 时，函数位于 \(\P_1\) 处，而 \(t=1\) 时，函数位于 \(\P_2\) 处，且它是连接 \(\P_1,\P_2\) 的一条直线。

参数

  • p0 (ndarray) – 一次 Bezier 曲线的起点

  • p1 (ndarray) – 一次 Bezier 曲线的终点

从起点和终点构造一次 Bezier 曲线

class pychai.base.Cubic(p0, p1, p2, p3)¶

三次 Bezier 曲线的形式为：

\[\r_3(t) = (1-t)^3\p_1 + 3(1-t)^2t\p_2 + 3(1-t)t^2\p_3 + t^3\p_4\]
参数

  • p0 (ndarray) – 三次 Bezier 曲线的起点

  • p1 (ndarray) – 三次 Bezier 曲线的第一控制点

  • p2 (ndarray) – 三次 Bezier 曲线的第二控制点

  • p1 – 三次 Bezier 曲线的终点

从起点和终点构造一次 Bezier 曲线

class pychai.base.Compound(name, operator, firstChild, secondChild, mix, **kwargs)¶

复合字

参数

  • name (str) – 复合字的名称

  • operator (str) – 复合字构字的运算符

  • firstChild (Character) – 复合字构字的第一个操作数，可以是部件也可以是复合字

  • secondChild (Character) – 复合字构字的第二个操作数，可以是部件也可以是复合字

  • mix (int) – 如果第二个操作数的所有笔画并不都在第一个操作数之后，则该整数代表实际书写顺序中，第二个操作数的第一笔是在第一个操作数的第几笔之后开始写的。例如，「区」分解为「匚」和「乂」时，mix = 1
